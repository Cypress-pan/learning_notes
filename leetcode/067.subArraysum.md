# 67.和为k的子数组

## 题目描述

给你一个整数数组 nums 和一个整数 k ，请你统计并返回 该数组中和为 k 的子数组的个数 。

子数组是数组中元素的连续非空序列。

**示例 1：**

输入：nums = [1,1,1], k = 2

输出：2

**示例 2：**

输入：nums = [1,2,3], k = 3

输出：2

## 解题思路

算法思路

前缀和：current_sum 表示从数组开始到当前位置的元素和。

关键观察：如果存在两个位置 i 和 j（i < j），使得 prefix_sum[j] - prefix_sum[i] = k，那么从位置 i+1 到 j 的子数组和就是 k。

变形：prefix_sum[j] - k = prefix_sum[i]，所以我们只需要查找是否存在前缀和为 current_sum - k 的位置。

哈希表：用哈希表记录每个前缀和出现的次数，这样可以在 O(1) 时间内查找。

示例

对于 nums = [1,1,1], k = 2：

i=0: current_sum=1, 查找 1-2=-1（不存在），count=0

i=1: current_sum=2, 查找 2-2=0（存在，次数1），count=1

i=2: current_sum=3, 查找 3-2=1（存在，次数1），count=2

返回 2，正确。

时间复杂度

O(n)，只需要遍历一次数组。

空间复杂度

O(n)，哈希表最多存储 n 个不同的前缀和。

## 代码示例
```python
def subarraySum(self, nums, k):
    """
    :type nums: List[int]
    :type k: int
    :rtype: int
    """
    # 使用哈希表存储前缀和出现的次数
    # key: 前缀和, value: 该前缀和出现的次数
    prefix_sum_count = {0: 1}  # 初始化，前缀和为0出现1次（空数组）
    
    current_sum = 0  # 当前前缀和
    count = 0       # 结果计数
    
    for num in nums:
        current_sum += num
        
        # 如果 current_sum - k 在哈希表中存在
        # 说明存在一个前缀和为 current_sum - k 的位置
        # 从该位置到当前位置的子数组和就是 k
        if current_sum - k in prefix_sum_count:
            count += prefix_sum_count[current_sum - k]
        
        # 更新当前前缀和的出现次数
        prefix_sum_count[current_sum] = prefix_sum_count.get(current_sum, 0) + 1
    
    return count
