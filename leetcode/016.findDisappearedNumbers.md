# 16.找到所有数组中消失的数字（findDisappearedNumbers）

## 题目描述

给你一个含 n 个整数的数组 nums ，其中 nums[i] 在区间 [1, n] 内。请你找出所有在 [1, n] 范围内但没有出现在 nums 中的数字，并以数组的形式返回结果。

 **示例 1：**

输入：nums = [4,3,2,7,8,2,3,1]

输出：[5,6]

**示例 2：**

输入：nums = [1,1]

输出：[2]

## 解题思路

核心原理：数字和索引的映射关系和正负号的存在性。

过程：遍历数组，将每个数字-1作为索引，寻找索引对应的数并将他变成负数，最后没有变成负数的数字说明不存在对应他的索引，根据索引变换回对应的数字即为缺少的数字。

为什么正数就是不存在的？

原始数组有 n 个元素，数字范围是 1~n

每个出现的数字，都会把对应索引位置标记为负数 → 出现过的数字，其对应索引一定是负数

反过来：

→ 如果某个索引还是正数

→ 说明没有任何数字来标记过它

→ 即该索引对应的数字从未出现

→ 所以该数字缺失！

## 代码示例
```python

class Solution(object):
    def findDisappearedNumbers(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        for num in nums:
            index=abs(num)-1
            if nums[index]>0:
                nums[index]=-nums[index]
        res=[]
        for i in range(len(nums)):
            if nums[i]>0:
                res.append(i+1)
        return res
