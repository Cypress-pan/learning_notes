# 18.二叉树的直径

## 题目描述

给你一棵二叉树的根节点，返回该树的直径 。

二叉树的直径是指树中任意两个**叶子节点**之间最长路径的长度。这条路径可能经过也**可能不经过根节点 root** 。

两节点之间路径的 长度 由它们之间边数表示。

## 解题思路

由于题目已经给出提示，二叉树的直径指数中任意两个节点之间最长路径的长度，所以此题所求的应该是两个节点之间边数的最大值。

这里要厘清树的高度和直径的区别：

树的高度：从当前节点到「最远叶子节点」的边数（或节点数，这里统一按节点数算，不影响核心）。

节点的直径贡献：以当前节点为「转折点」，向左走到头的高度 + 向右走到头的高度（也就是穿过这个节点的最长路径）。

整棵树的直径：所有节点的「直径贡献」中的最大值（不管这个节点是不是根节点）

两个叶子节点之间的边数=所有节点中到这两个叶子结点的最大长度。

## 代码示例
```python
def diameterOfBinaryTree(self, root):
        """
        :type root: Optional[TreeNode]
        :rtype: int
        """
        if not root:
            return 0
        
        # 用实例变量保存全局最大直径 嵌套函数需要和外层类方法共享一个 “全局” 变量，且这个变量需要跨递归调用保留值。
        self.max_diameter = 0
        
        # 嵌套函数不需要 self 参数！
        #self：是 Python 类的实例方法的第一个参数，它的作用是指向当前类的实例对象，只有在「类的方法」中才需要显式写（或隐式传递）。
#嵌套函数：是定义在另一个函数（这里是 diameterOfBinaryTree 这个类方法）内部的普通函数，它不属于类的方法，只是一个 “内部工具函数”。
        def height(node):
            if not node:
                return 0
            left_height = height(node.left)
            right_height = height(node.right)
            
            # 关键：计算当前节点的直径（直接贡献），并更新全局最大值
            current_diameter = left_height +right_height
            if current_diameter > self.max_diameter:
                self.max_diameter =current_diameter
            
            # 返回当前节点的高度
            return max(left_height, right_height) + 1
        
        # 调用 height 函数，会自动遍历所有节点并更新 max_diameter
        height(root)
        
        # 返回全局最大直径（而不是只返回根节点的直径）
        return self.max_diameter
