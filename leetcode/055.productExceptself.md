# 55.除了自身以外数组的最大乘积

## 代码示例
```python
def productExceptSelf(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        #利用索引左侧所有数字的乘积和右侧所有数字的乘积相乘得到答案
        #初始化两个空数组L,R.对于给定索引i,L[i]代表的是i左侧所有数字的乘积，R[i]代表右侧所有数字的乘积
        #用两个循环来填充L R数组。L[i]=L[i-1]*nums[i-1] R[i]=R[i+1]*nums[i+1]
        #当填充完成后我们只需要在输入数组上迭代，且索引i处的值为L[i]*R[i]

        n=len(nums)
        L,R,answer=[0]*n,[0]*n,[0]*n
        L[0]=1
        R[n-1]=1
        for i in range(1,n):
            L[i]=L[i-1]*nums[i-1]
        for i in reversed(range(n-1)):
            R[i]=R[i+1]*nums[i+1]
        for i in range(n):
            answer[i]=L[i]*R[i]
        return answer
