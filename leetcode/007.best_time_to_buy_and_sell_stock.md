# 7. 买卖股票的最佳时机（Best Time to Buy and Sell Stock）

##  题目描述

给定一个数组 `prices`，其中 `prices[i]` 表示某支股票第 `i` 天的价格。

你只能选择**某一天**买入这只股票，并选择在**未来的某一天**卖出（只能买卖一次）。

设计一个算法来计算你所能获取的**最大利润**。如果你不能获取任何利润，返回 `0`。

### 示例

**示例 1：**

输入: `prices = [7,1,5,3,6,4]`

输出: 5

解释: 在第 2 天（价格=1）买入，在第 5 天（价格=6）卖出，利润 = 6-1 = 5。
注意不能在第 1 天买入第 2 天卖出（7-1=6），因为必须先买后卖。

**示例二：**

输入: `prices = [7,6,4,3,1]`

输出: 0

解释: 价格一直下跌，没有交易完成，最大利润为 0。

## 解题思路

### 核心思想

**贪心算法**：对于每一天，如果当天卖出，最大利润 = 当天价格 - 之前最低价格。

所以我们只需要：
1. 遍历过程中维护一个**历史最低价**（买入价）
2. 每天计算**当天卖出**的利润
3. 更新**最大利润**

**暴力求解在此题运行超时**

## 代码示例
```python
def maxProfit(self,prices):
    min_price=float('inf')#定义为正无穷大
    max_price=0
    for price in prices:
        min_price=min(min_price,price)#更新最小价格
        max_price=max(price-min_price,max_price)#更新最大价格
    return max_price
```
## 相关补充

### 贪心算法

核心思想：每一步都做出当前看来最优的选择，希望通过局部最优达到全局最优。（不回溯）
