# 60.买卖股票的最佳时机

## 代码示例
```python
def maxProfit(self, prices):
        """
        :type prices: List[int]
        :rtype: int
        """
        if not prices:
            return 0
        #dp[i][0]第i-1天持有的最大利润 dp[i][1]第i天不持有股票的最大利润
        #对于第i天不持有股票，可能今天刚卖或者昨天就不持有dp[i][1]=max(dp[i-1][1],dp[i-1][0]+prices[i])
        #第i天持有股票，今天刚买入或者昨天就有dp[i][0]=max(dp[i-1][0],dp[i-2][1]-prices[i])
        n=len(prices)
        dp=[[0]*n for _ in range(0,2)]
        dp[0][0]=-prices[0]
        dp[0][1]=0
        dp[1][0]=max(dp[0][0],dp[0][1]-prices[1])
        dp[1][1]=max(dp[0][1],dp[0][0]+prices[1])
        for i in range(2,n):
            dp[i][1]=max(dp[i-1][1],dp[i-1][0]+prices[i])
            dp[i][0]=max(dp[i-1][0],dp[i-2][1]-prices[i])
        return dp[n-1][1]
