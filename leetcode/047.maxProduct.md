# 47.乘积最大数组

## 题目描述

给你一个整数数组 nums ，请你找出数组中乘积最大的非空连续 子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

测试用例的答案是一个 32-位 整数。

请注意，一个只包含一个元素的数组的乘积是这个元素的值。

**示例 1:**

输入: nums = [2,3,-2,4]

输出: 6

解释: 子数组 [2,3] 有最大乘积 6。

**示例 2:**

输入: nums = [-2,0,-1]

输出: 0

解释: 结果不能为 2, 因为 [-2,-1] 不是子数组。

## 解题思路

这道题乍一看的时候我以为是动态规划，但是一般来说，对于加法，因为加法具有单调性：前面的和越大，加上当前数通常也越大，倘若当前是负数，也可以直接跳过或继续寻找；

但是这道题是乘法，根据负负得正，对于一个很小的负数，乘上当前的正数后可能变成很大的数，所以前一步最坏的情况可能在这步变成最好的情况。

要解决这个问题，我们需要维护两个状态：当前最大值 (max_dp)：以当前位置结尾的子数组的最大乘积。当前最小值 (min_dp)：以当前位置结尾的子数组的最小乘积（为了应对负数）。

假设当前数字是 num，前一个位置的最大值是 prev_max，最小值是 prev_min。当前位置的最大值 curr_max 是以下三者的最大值：

num（重新开始，不接前面的）

num * prev_max（正数接正数，或负数接负数）

num * prev_min（负数接最小负数，负负得正）

同理，当前位置的最小值 curr_min 是这三者的最小值。

```python
    def maxProduct(nums):
    # #dp[i]长度为i的数组中子数组的最大乘积
        # #dp[0]=0,dp[1]=nums[0],dp[2]=max(nums[1],nums[1]*dp[0])
        # #遍历顺序，从前往后
        # if not nums:
        #     return 0
        # n=len(nums)
        # dp=[0]*n
        # dp[0]=0
        # dp[1]=nums[0]
        # for i in range(2,n):
        #     dp[i]=max(nums[i-1],nums[i-1]*dp[i-1])
        # return dp[n]
    
        # 1. 先判空！防止下面 nums[0] 报错
        if not nums:
            return 0
        
        # 2. 初始化
        maxtofar = nums[0]
        mintofar = nums[0]
        res = nums[0]
        n = len(nums)
        
        for i in range(1, n):
            # 3. 同时更新，避免变量覆盖
            # Python 允许这样写：右边的 maxtofar 和 mintofar 都是上一轮的值
            maxtofar, mintofar = (
                max(nums[i], maxtofar * nums[i], mintofar * nums[i]),
                min(nums[i], maxtofar * nums[i], mintofar * nums[i])
            )
            
            # 4. 更新结果
            res = max(res, maxtofar)
            
        return res
