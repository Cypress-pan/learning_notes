# 69.单词搜索

## 题目描述

给定一个 m x n 二维字符网格 board 和一个字符串单词 word 。如果 word 存在于网格中，返回 true ；否则，返回 false 。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

示例 1：

输入：board = [['A','B','C','E'],['S','F','C','S'],['A','D','E','E']], word = "ABCCED"

输出：true

示例 2：

输入：board = [['A','B','C','E'],['S','F','C','S'],['A','D','E','E']], word = "SEE"

输出：true

示例 3：

输入：board = [['A','B','C','E'],['S','F','C','S'],['A','D','E','E']], word = "ABCB"

输出：false

## 解题思路

先检查边界和字符是否匹配 -> 标记当前格子已访问 -> 向四个方向递归 -> 递归结束后恢复现场（取消标记）

## 代码示例
```python
def exist(self, board, word):
    """
    :type board: List[List[str]]
    :type word: str
    :rtype: bool
    """
    if not board or not board[0]:
        return False
    
    rows, cols = len(board), len(board[0])
    # 创建一个和 board 一样大的 visited 数组，初始都是 False
    visited = [[False] * cols for _ in range(rows)]
    
    # 1. 从左上角开始遍历整个网格
    for i in range(rows):
        for j in range(cols):
            # 2. 如果遇到单词的开头字母，开始深度搜索
            if board[i][j] == word[0]:
                if self.dfs(board, word, i, j, 0, visited):
                    return True
    return False

def dfs(self, board, word, r, c, index, visited):
    """
    r, c: 当前所在的行和列
    index: 当前正在匹配 word 的第几个字符
    """
    # 终止条件：如果 index 等于单词长度，说明所有字符都找到了
    if index == len(word):
        return True
    
    # 边界检查：越界、当前字符不匹配、或者已经访问过
    if (r < 0 or r >= len(board) or c < 0 or c >= len(board[0]) or 
        visited[r][c] or board[r][c] != word[index]):
        return False
    
    # 3. 标记当前格子为已访问
    visited[r][c] = True
    
    # 4. 按照 上、右、下、左 的顺序继续搜索下一个字符
    # 只要有一个方向返回 True，整体就是 True
    found = (self.dfs(board, word, r-1, c, index+1, visited) or  # 上
             self.dfs(board, word, r, c+1, index+1, visited) or  # 右
             self.dfs(board, word, r+1, c, index+1, visited) or  # 下
             self.dfs(board, word, r, c-1, index+1, visited))    # 左
    
    # 5. 回退（回溯）：如果四个方向都没找到，或者为了寻找其他路径，
    # 必须把当前格子恢复为未访问状态
    visited[r][c] = False
    
    return found
