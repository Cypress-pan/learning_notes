# 5.二叉树的中序遍历

## 问题描述
给定一个二叉树的根节点 `root `，返回 它的中序遍历 。

## 代码示例
```python
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        """
        二叉树的中序遍历（递归实现）
        :param root: 根节点
        :return: 中序遍历结果列表
        """
        res = []
        if root:
            res.extend(self.inorderTraversal(root.left))   # 左子树
            res.append(root.val)                           # 根节点
            res.extend(self.inorderTraversal(root.right))  # 右子树
        return res
