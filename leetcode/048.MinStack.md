# 48.最小栈

## 题目描述

设计一个支持 push ，pop ，top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类:

• MinStack() 初始化堆栈对象。

• void push(int val) 将元素val推入堆栈。

• void pop() 删除堆栈顶部的元素。

• int top() 获取堆栈顶部的元素。

• int getMin() 获取堆栈中的最小元素。 

**示例 1:**

输入：

["MinStack","push","push","push","getMin","pop","top","getMin"]

[[],[-2],[0],[-3],[],[],[],[]]

输出：

[null,null,null,null,-3,null,0,-2]

解释：

MinStack minStack = new MinStack();

minStack.push(-2);

minStack.push(0);

minStack.push(-3);

minStack.getMin();   --> 返回 -3.

minStack.pop();

minStack.top();      --> 返回 0.

minStack.getMin();   --> 返回 -2.

class MinStack:
    def __init__(self):
        self.stack=[]
        self.min_stack=[]
#最小栈用来存储当前最小的值
    def push(self,val):
        if val<=self.min_stack[-1] or not self.min_stack:
            self.min_stack.append(val)
        self.stack.append(val)

    def pop():
        if not self.stack:
            return None
        value=self.stack.pop()
        if value==min_stack[-1]:
            self.min_stack.pop()
    def top():
        return self.stack[-1]
    def getMin():
        return self.min_stack[-1]

我还写了一种用链表存储的写法，但是如果pop的时最小元素，会产生O(N)的时间复杂度去寻找新的最小值，所以不是最优的。

```python
class LinkNode:
    def __init__(self, val):
        self.val = val
        self.pre = None
        self.next = None

class MinStack(object):
    def __init__(self):
        # 初始化一个空的链表
        self.head = None
        self.tail = None
        # 最小值作为类的属性，而不是全局变量
        self.min_val = float('inf') 

    def push(self, val):
        """
        :type val: int
        :rtype: None
        """
        new_node = LinkNode(val)
        
        # 1. 更新最小值
        if val < self.min_val:
            self.min_val = val
            
        # 2. 链表插入操作 (尾插法)
        if not self.head: # 如果是空链表
            self.head = new_node
            self.tail = new_node
        else:
            self.tail.next = new_node
            new_node.pre = self.tail
            self.tail = new_node # 更新 tail 指针

    def pop(self):
        """
        :rtype: None
        """
        if not self.tail: # 栈为空
            return
        
        # 1. 获取要删除节点的值
        val_to_remove = self.tail.val
        
        # 2. 链表删除操作
        if self.head == self.tail: # 只有一个节点
            self.head = None
            self.tail = None
        else:
            self.tail = self.tail.pre
            self.tail.next = None
            
        # 3. 关键难点：更新最小值
        # 如果删掉的是最小值，我们需要重新遍历链表找新的最小值
        # 注意：这会导致 pop 操作在最坏情况下变成 O(N)
        if val_to_remove == self.min_val:
            self._update_min()

    def top(self):
        """
        :rtype: int
        """
        if self.tail:
            return self.tail.val
        return None

    def getMin(self):
        """
        :rtype: int
        """
        return self.min_val
    
    def _update_min(self):
        """辅助函数：重新计算最小值"""
        if not self.head:
            self.min_val = float('inf')
            return
            
        curr = self.head
        self.min_val = curr.val
        while curr:
            if curr.val < self.min_val:
                self.min_val = curr.val
            curr = curr.next
