# 64.字符串解码

## 题目描述

给定一个经过编码的字符串，返回它解码后的字符串。

编码规则为: k[encoded_string]，表示其中方括号内部的 encoded_string 正好重复 k 次。注意 k 保证为正整数。

你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。

此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数 k ，例如不会出现像 3a 或 2[4] 的输入。

测试用例保证输出的长度不会超过 105。

示例 1：

输入：s = "3[a]2[bc]"

输出："aaabcbc"

示例 2：

输入：s = "3[a2[c]]"

输出："accaccacc"

示例 3：

输入：s = "2[abc]3[cd]ef"

输出："abcabccdcdcdef"

## 解题思路

双栈法，一个存数字一个存字母，

遇到数字：存入数字栈。

遇到 [：把当前的字符串存入字母栈（作为前缀），然后清空当前字符串。

遇到 ]：从数字栈弹出次数，从字母栈弹出前缀，把当前字符串复制 N 次，然后拼上前缀。

普通字符：加到当前字符串里。

例：

🌰 举个例子：3[a2[c]]

3: current_num = 3

[: 压栈！num_stack=[3], str_stack=['']。重置 current_num=0, current_str=''

a: current_str = 'a'

2: current_num = 2

[: 压栈！num_stack=[3, 2], str_stack=['', 'a']。重置 current_num=0, current_str=''

c: current_str = 'c'

]: 出栈！count=2, prev_str='a'。current_str = 'a' + 'c'*2 = 'acc'

]: 出栈！count=3, prev_str=''。current_str = '' + 'acc'*3 = 'accaccacc'

## 代码示例
```python
def decodeString(self, s):
        num_stack = []  # 存数字
        str_stack = []  # 存字符串（存的是遇到 [ 之前的前缀）
        
        current_str = "" # 当前正在构建的字符串
        current_num = 0  # 当前正在解析的数字
        
        for char in s:
            if char.isdigit():
                # 处理多位数，比如 "12[a]"
                current_num = current_num * 10 + int(char)
                
            elif char == '[':
                # 遇到左括号，把当前的数字和字符串都压栈
                num_stack.append(current_num)
                str_stack.append(current_str)
                
                # 重置，准备处理括号里的新内容
                current_num = 0
                current_str = ""
                
            elif char == ']':
                # 遇到右括号，开始解码
                # 1. 弹出次数
                count = num_stack.pop()
                # 2. 弹出之前的前缀
                prev_str = str_stack.pop()
                # 3. 拼接：前缀 + 当前字符串 * 次数
                current_str = prev_str + current_str * count
                
            else:
                # 普通字母
                current_str += char
                
        return current_str
