# 46.排序链表

## 题目描述

给你链表的头结点 head ，请将其按 升序 排列并返回 排序后的链表 。

**示例 1：**

输入：head = [4,2,1,3]

输出：[1,2,3,4]

**示例 2：**

输入：head = [-1,5,3,4,0]

输出：[-1,0,3,4,5]

**示例 3：**

输入：head = []

输出：[]

## 解题思路

本题采用分治法的思想，从链表中间节点分成两端对左右两端分别递归调用函数，别忘了断开的时候把slow.next设置成none。当达到终止条件，即只剩一个节点时，把节点的next指针断开，return。

接下来就是合并了。采取传统的思想，比较大小，小的加入，最后处理剩下的节点。

## 代码示例
```python
class Solution(object):

    def merge(self,l,r):
        dummy=ListNode(0)
        cur=dummy
        while l and r:
            if l.val<r.val:
                cur.next=l
                cur=cur.next
                l=l.next
            else:
                cur.next=r
                cur=cur.next
                r=r.next
        cur.next=l if l else r
        return dummy.next

    def sortList(self, head):
        """
        :type head: Optional[ListNode]
        :rtype: Optional[ListNode]
        """
        if not head or not head.next:
            return head
        slow=head
        fast=head.next
        while fast and fast.next:
            slow=slow.next
            fast=fast.next.next
        mid=slow.next
        slow.next=None#断开
        left=self.sortList(head)
        right=self.sortList(mid)
        return self.merge(left,right)
