# 51.数组中第K个最大的元素

## 题目描述

给定整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。

请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。

你必须设计并实现时间复杂度为 O(n) 的算法解决此问题。

**示例 1:**

输入: [3,2,1,5,6,4], k = 2

输出: 5

**示例 2:**

输入: [3,2,3,1,2,4,5,5,6], k = 4

输出: 4

## 解题思路

本题考查的是快速选择算法。从数组中随机选取一个数作为基准，将小于基准和大于基准的数分别放置在基准右，左两边，中间是等于基准的数组。

若k小于左边数组长度则递归选择左边子数组；

若k小于等于左边数组长度和中间数组长度之和，则直接返回pivot；

否则递归调用右边数组。

## 代码示例
```python

def findKthLargest(self, nums, k):
        # 1. 选基准值（这里简单选第一个，也可以随机选）
        pivot = nums[0]
        
        # 2. 分成三部分：
        # left: 比 pivot 大的数 (因为我们找第 K 大，所以大的放前面)
        # mid: 等于 pivot 的数
        # right: 比 pivot 小的数
        left = [x for x in nums if x > pivot]
        mid = [x for x in nums if x == pivot]
        right = [x for x in nums if x < pivot]
        
        # 3. 判断 k 落在哪个区间
        if k <= len(left):
            # 如果 k 在左边，说明第 k 大在“比 pivot 大”的数里
            return self.findKthLargest(left, k)
        elif k <= len(left) + len(mid):
            # 如果 k 在中间，说明 pivot 就是答案
            return pivot
        else:
            # 如果 k 在右边，说明第 k 大在“比 pivot 小”的数里
            # 注意：k 要减去左边和中间的数量，变成新的相对位置
            return self.findKthLargest(right, k - len(left) - len(mid))


##用双指针原地分区
class Solution(object):
    def findKthLargest(self, nums, k):
        def partition(left, right, pivot_idx):
            pivot = nums[pivot_idx]
            # 1. 将 pivot 移到末尾
            nums[pivot_idx], nums[right] = nums[right], nums[pivot_idx]
            
            # 2. 将所有大于 pivot 的数移到左边
            store_idx = left
            for i in range(left, right):
                if nums[i] > pivot: # 注意是 >，因为找第 K 大
                    nums[store_idx], nums[i] = nums[i], nums[store_idx]
                    store_idx += 1
            
            # 3. 将 pivot 移回中间
            nums[right], nums[store_idx] = nums[store_idx], nums[right]
            return store_idx

        def select(left, right, k_smallest):
            if left == right:
                return nums[left]
            
            import random
            pivot_idx = random.randint(left, right)
            pivot_idx = partition(left, right, pivot_idx)
            
            if k_smallest == pivot_idx:
                return nums[k_smallest]
            elif k_smallest < pivot_idx:
                return select(left, pivot_idx - 1, k_smallest)
            else:
                return select(pivot_idx + 1, right, k_smallest)
        
        # 第 k 大 = 第 (k-1) 小（从0开始索引）
        return select(0, len(nums) - 1, k - 1)
