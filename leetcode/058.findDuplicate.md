# 58.寻找重复数

## 题目描述

给定一个包含 n + 1 个整数的数组 nums ，其数字都在 [1, n] 范围内（包括 1 和 n），可知至少存在一个重复的整数。

假设 nums 只有 一个重复的整数 ，返回 这个重复的数 。

你设计的解决方案必须 不修改 数组 nums 且只用常量级 O(1) 的额外空间。

**示例 1：**

输入：nums = [1,3,4,2,2]

输出：2

**示例 2：**

输入：nums = [3,1,3,4,2]

输出：3

**示例 3 :**

输入：nums = [3,3,3,3,3]

输出：3

## 解题思路

**方法一：判圈算法**

**判圈算法是检测链表是否有环的算法**。我们对nums数组建图，每个索引i通向他的value对应的索引位置。由于存在重复的数字target，因此target这个位置一定起码有两条指向他的边，因此整张图一定存在环，且我们要找到的target就是这个环的入口。

我们先设置指针slow，fast，慢指针每次走一步，快指针每次走两步，根据判圈算法，两个指针在有环的情况下一定会相遇，此时我们再将slow放置起点0，两个指针每次移动一步，相遇点就是答案。

value 1 4 6 6 6 2 3

index 0 1 2 3 4 5 6

(1) slow=0 fast=0 slow move first and arrive at index 1(value 4);fast move second and reach index 1 because fast move 2 inch so it reach index 4(value[1]=4)

(2) slow move 1 inch and reach index 4;fast move and reach index 6(value=3) and reach index 3(value6)

(3) value[slow]==value[fast] so they meet. let slow=0

(4) slow and fast move one step each time and when they meet second return 

## 代码示例
```python
def finDuplicate(nums):
  if not nums:
    return -1
  slow,fast=0,0
  while True:
    slow=nums[slow]
    fast=nums[nums[fast]]
    if slow==fast:
      break
  slow=0
  while slow!=fast:
    slow=nums[slow]
    fast=nums[fast]
  return slow
```
## 方法二：二进制

将所有数二进制展开按位考虑如何找出重复的数，如果我们能确定重复数每一位是1 还是0 就可以按位还原出重复的数是什么。

考虑第i位，我们记nums数组中二进制展开后第i位为1的数有x个，数字[1,n]这n个数二进制展开后第i位为1的数有y个，那么重复的数第i位为1当且仅当x>y。

例如：

         1  3  4  2  2  x  y

第0位    1  1  0  0  0  2  2

第1位    0  1  0  1  1  3  2

第2位    0  0  1  0  0  1  1

不难看出，只有第1位x>y，按位还原 target=010=2

```python
def findDuplicate(self, nums):
    """
    :type nums: List[int]
    :rtype: int
    """
    # 题目隐含条件：数字范围是 1 到 n，数组长度是 n + 1
    # 我们需要确定重复数字的二进制每一位
    n = len(nums) - 1 
    target = 0
    
    # 假设整数是 32 位的，我们遍历每一位（0 到 31）
    # 实际上只需要遍历到 log2(n) 即可，写 32 是为了通用性
    for i in range(32):
        x = 0 # 统计 nums 数组中第 i 位为 1 的个数
        y = 0 # 统计 [1, n] 范围内第 i 位为 1 的个数
        
        # 1. 统计 nums 数组中第 i 位为 1 的数量 (x)
        for num in nums:
            # (num >> i) & 1 用于取出 num 的第 i 位
            if (num >> i) & 1:
                x += 1
        
        # 2. 统计 1 到 n 中第 i 位为 1 的数量 (y)
        for k in range(1, n + 1):
            if (k >> i) & 1:
                y += 1
        
        # 3. 核心逻辑：如果 x > y，说明多出来的那个数（重复数）在第 i 位是 1
        if x > y:
            # 将 target 的第 i 位设为 1
            # 1 << i 相当于 2 的 i 次方
            target |= (1 << i)
            
    return target
