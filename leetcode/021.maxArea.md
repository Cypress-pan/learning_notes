# 21.盛最多水的容器

## 题目描述

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

说明：你不能倾斜容器。

## 解题思路

因为水的面积是底乘高，所以最初先让底最长，即左右两指针分别在数组两头，因为水的高取决于较低的那个边，比较左右指针对应的高，较小的往中间移动，直到两指针相遇。

## 代码示例
```python
def maxArea(self,height):
    left,right=0,len(height)-1
    res=0
    while left<right:
        width=right-left
        cur_height=height[left] if height[left]<height[right] else height[right]
        area=width*cur_height
        res=max(res,area)
        if height[left]<height[right]:
            left+=1
        else right-=1
    return res
