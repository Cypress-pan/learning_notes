# 20.最长回文子串

## 题目描述

给你一个字符串 s，找到 s 中最长的 回文 子串。

 **示例 1：**

输入：s = "babad"

输出："bab"

解释："aba" 同样是符合题意的答案。

**示例 2：**

输入：s = "cbbd"

输出："bb"

## 解题思路

本题采用中心扩展法。从字符串的中间向两边扩展，找到最长的回文字符串。对于题目给定的字符串，我们不知道中心是哪个字符，在什么地方，所以需要从字符串开头遍历，对于每个字符，分奇数或偶数进行讨论，遍历结束后最长的即为所求。
