# 40.最长连续序列

## 题目描述

给定一个未排序的整数数组 nums ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 O(n) 的算法解决此问题。

**示例 1：**

输入：nums = [100,4,200,1,3,2]

输出：4

解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。

**示例 2：**

输入：nums = [0,3,7,2,5,8,4,6,0,1]

输出：9

**示例 3：**

输入：nums = [1,0,1,2]

输出：3

## 解题思路

用哈希集合快速判断 “某个数是否存在”，只从连续序列的 “起点” 开始计数，避免重复计算，实现 O (n) 效率。

具体实现：从头开始遍历数组，对于当前数，判断num-1是否在集合内，不在说明是连续序列的起点，对于num-1在集合内的数，不予处理，因为这样会重复计算。

从连续序列起点开始计数，若num+1在集合内，更新目前长度和当前数。同步更新最大长度。

## 代码示例
```python
def longestconsecutive(nums):
    if not nums:
        return []
    max_len=0
    nums_set=set(nums)
    for num in nums:
        if num-1 not in nums_set:
            #说明是序列的起点
            current_num=num
            current_len=1
            while num+1 in nums_set:
                current_len+=1
                current_num+=1
            #找完当前的序列，更新
            max_len=max(max_len,current_len)
    return max_len
