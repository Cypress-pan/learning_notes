# 26.下一个排列

## 解题思路

以[1,3,5,4,2]为例：

第一步：从右向左找升序对，找第一个nums[i]<nums[i+1]的位置

[1, 3, 5, 4, 2]
     
    ↑
      
    i=1 (3 < 5) ✓

第二步：在右边找比nums[i]大的最小数

右边部分[5,4,2]

找比3大的最小数 4

第三步：交换+翻转右边

交换3 4：[1,4,5,3,2]

反转i后面的部分[1,4,2,3,5]

为什么要反转呢？

交换后：[1, 4, 5, 3, 2]
             
              ↑
            
            这部分还是降序 [5,3,2]

降序是"最大"排列，反转为升序才是"最小"排列

→ [1, 4, 2, 3, 5] 才是下一个最小的！

## 代码示例
```python
class Solution:
    def nextPermutation(self, nums: List[int]) -> None:
        n = len(nums)
        
        # 第一步：从右找第一个升序对 nums[i] < nums[i+1]
        i = n - 2
        while i >= 0 and nums[i] >= nums[i + 1]:
            i -= 1
        
        # 第二步：如果找到了，在右边找比 nums[i] 大的最小数
        if i >= 0:
            j = n - 1
            while nums[j] <= nums[i]:  # 从右找第一个比它大的
                j -= 1
            nums[i], nums[j] = nums[j], nums[i]  # 交换
        
        # 第三步：反转 i 后面的部分（不管找没找到都要反转！）
        # 没找到说明整个降序，反转后就是最小排列
        left, right = i + 1, n - 1
        while left < right:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
