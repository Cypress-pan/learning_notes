# 25.括号的生成

## 题目描述

数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

 **示例 1：**

输入：n = 3

输出：["((()))","(()())","(())()","()(())","()()()"]

**示例 2：**

输入：n = 1

输出：["()"]

## 解题思路

本题沿用上一题的回溯法。且满足三个条件：

1、左括号与右括号数之和等于2N

2、左括号数应该小于等于n

3、右括号数在小于左括号数的时候才可以添加右括号

## 代码示例
```python

class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        result = []
        
        def backtrack(current, left, right):
            # 终止条件：用完所有括号
            if len(current) == 2 * n:
                result.append(current)
                return
            
            # 选择1：放左括号（只要还有剩余）
            if left < n:
                backtrack(current + "(", left + 1, right)
            
            # 选择2：放右括号（必须左括号更多）
            if right < left:
                backtrack(current + ")", left, right + 1)
        
        backtrack("", 0, 0)
        return result
