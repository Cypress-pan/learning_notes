# 36.不同路径

## 题目描述

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为 “Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？

**示例 1：**

输入：m = 3, n = 7

输出：28

**示例 2：**

输入：m = 3, n = 2

输出：3

解释：

从左上角开始，总共有 3 条路径可以到达右下角。

1. 向右 -> 向下 -> 向下

2. 向下 -> 向下 -> 向右

3. 向下 -> 向右 -> 向下

## 解题思路

核心考点：动态规划

(i,j)意为从原点到(i,j)的路径数。

这道题可以用dfs思想但是会超时。

代码示例如下
```python
def uniquePaths(self, m: int, n: int) -> int:
    # 不用 visited！因为只能右/下走，不会走回头路（无环）
    
    def dfs(row, col):
        # 越界检查
        if row >= m or col >= n:
            return 0
        # 到达终点
        if row == m - 1 and col == n - 1:
            return 1
        
        # 右 + 下，两条路的结果相加
        return dfs(row, col + 1) + dfs(row + 1, col)
    
    return dfs(0, 0)
```

## 代码示例
```python
def uniquePaths(self, m: int, n: int) -> int:
    dp = [[0] * n for _ in range(m)]
    #初始化第一行和第一列，因为到（i,j）的路径只有一种
    for i in range(n):
        dp[0][i] = 1
    for j in range(m):
        dp[j][0] = 1
    
    # 填表
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = dp[i-1][j] + dp[i][j-1]
    
    return dp[m-1][n-1]
