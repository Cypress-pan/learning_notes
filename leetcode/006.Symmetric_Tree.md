# 6. 对称二叉树 (Symmetric Tree)

## 题目描述

给你一个二叉树的根节点 `root` ，检查它是否轴对称。

**示例 1：**
输入：root = [1,2,2,3,4,4,3]
输出：true

**示例 2：**
输入：root = [1,2,2,null,3,null,3]
输出：false

## 解题思路

判断二叉树对称的本质：比较根节点的左右子树是否互为镜像。
镜像的条件：
- 两个根结点的值相等
- 左子树的左子树与右子树的右子树对称
- 左子树的右子树与右子树的左子树对称

    def isMirror(left,right):
        #都为空，对称
        if not left and not right:
            return True
        if not left or not right:
            return False
         #当前节点值相等，且左的左子树与右的右子树对称，左的右子树与右的左子树对称
         return (left.val==right.val and isMirror(left.left,right.right) and isMirror(left.right,right.left))
    if not root:
        return True
    return isMirror(root.left,root.right)
```
## 注意
不能直接中序遍历二叉树然后判断列表是不是回文！！！（我犯了这个错）因为[1,2,2,null,3,null,3]的中序遍历是[2,3,1,2,3]但是结构并不对称。
