# 2. 有效的括号 (Valid Parentheses)
基于**栈** + **字典**实现的有效括号验证，核心思路为「左括号入栈，右括号匹配栈顶」。

## 题目描述
给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。
有效字符串需满足：
1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 空字符串被认为是有效字符串。

## 解题思路
1. 初始化一个空栈，用于存储左括号。
2. 定义字典映射左括号到对应右括号（`{'[': ']', '{': '}', '(': ')'}`）。
3. 遍历字符串每个字符：
   - 若为左括号（字典的键），入栈；
   - 若为右括号，先检查栈是否为空（无左括号匹配则直接无效），再弹出栈顶左括号，判断其对应的右括号是否与当前字符一致；
4. 遍历结束后，栈必须为空（避免剩余未匹配的左括号）。

## 使用方法
### 运行核心代码
```python
class Solution(object):
    def isValid(self, s):
        """
        :type s: str
        :rtype: bool
        """
        # 初始化栈
        stack = []
        # 左括号到右括号的映射
        bracket_dict = {'[': ']', '{': '}', '(': ')'}
        
        # 遍历字符串每个字符
        for elem in s:
            # 若为左括号，入栈
            if elem in bracket_dict:
                stack.append(elem)
            # 若为右括号
            else:
                # 空栈保护 + 匹配栈顶左括号对应的右括号（你的思路优化）
                if not stack or bracket_dict[stack.pop()] != elem:
                    return False
        
        # 遍历结束后检查栈是否为空（避免剩余左括号）
        return len(stack) == 0
