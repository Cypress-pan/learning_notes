# 70.不同的二叉搜索树

## 题目描述

给你一个整数 n ，求恰由 n 个节点组成且节点值从 1 到 n 互不相同的 二叉搜索树 有多少种？返回满足题意的二叉搜索树的种数。

示例 1：

输入：n = 3

输出：5

示例 2：

输入：n = 1

输出：1

## 解题思路

状态定义：dp[i] 表示由 1 到 i 这 i 个数字能组成的不同二叉搜索树的数量。

状态转移：

我们要构建 i 个节点的树，可以依次选 1 到 i 作为根节点。

如果选 j 作为根节点：

左子树：包含 1 到 j-1，共 j-1 个节点，有 dp[j-1] 种。

右子树：包含 j+1 到 i，共 i-j 个节点，有 dp[i-j] 种。

所以以 j 为根的树有 dp[j-1] * dp[i-j] 种。

把所有 j 的情况加起来：dp[i] = sum(dp[j-1] * dp[i-j])，其中 j 从 1 到 i。

## 代码示例
```python
def numTrees(self, n):
    """
    :type n: int
    :rtype: int
    """
    # dp[i] 表示 1 到 i 个节点能组成的二叉搜索树数量
    dp = [0] * (n + 1)
    
    # 初始化
    dp[0] = 1  # 空树也是一种树
    dp[1] = 1  # 只有一个节点，一种树
    
    # 计算 dp[2] 到 dp[n]
    for i in range(2, n + 1):
        # 遍历 1 到 i 作为根节点
        for j in range(1, i + 1):
            # 左子树节点数: j-1
            # 右子树节点数: i-j
            # 当前根节点的树数量 = 左子树数量 * 右子树数量
            dp[i] += dp[j-1] * dp[i-j]
            
    return dp[n]
