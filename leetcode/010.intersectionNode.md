# 10.相交链表

## 解题思路

将两个链表右对齐，如果A比B长，就将A的指针移到和B开头对齐，反之亦然。然后分别比较两个链表的指针，如果相等则说明相交（注意这里的相等不仅仅是数值的相等，是结点的内存地址）。

## 代码示例
```python
# Definition for singly-linked list.
# class ListNode(object):
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution(object):
    def getIntersectionNode(self, headA, headB):
        """
        :type head1, head1: ListNode
        :rtype: ListNode
        """
        def length(self,head):
            pos=head
            count=0
            while pos:
                count+=1
                pos=pos.next
            return count
        #计算两个链表的长度
        numA=length(self,headA)
        numB=length(self,headB)
        posA=headA
        posB=headB
        skip_num=numA-numB if numA>numB else numB-numA
        #对齐指针开头
        if numA>numB:
            while skip_num:
                posA=posA.next
                skip_num-=1
        else:
            while skip_num:
                posB=posB.next
                skip_num-=1
        while posA and posB:
            if posA==posB:#找到
                return posA
            posA=posA.next
            posB=posB.next
        return None
