# 17.汉明距离（hammingDistance）

## 题目描述

两个整数之间的 汉明距离 指的是这两个数字对应二进制位不同的位置的数目。

给你两个整数 x 和 y，计算并返回它们之间的汉明距离。

## 解题思路

对于两个二进制数，逐位比较，0^0=0，0^1=1，所以不同的位异或等于1，数出异或运算后1的个数就是距离。

## 代码示例
```python
def hammingDistance(self, x, y):
        """
        :type x: int
        :type y: int
        :rtype: int
        """
        o=x^y
        return bin(o).count('1')
