# 23.无重复字符的最长子串

## 题目描述

给定一个字符串 s ，请你找出其中不含有重复字符的 最长 子串 的长度。

 ** 示例 1:**

输入: s = "abcabcbb"

输出: 3 

解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。

**示例 2:**

输入: s = "bbbbb"

输出: 1

解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

**示例 3:**

输入: s = "pwwkew"

输出: 3

解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。

## 解题思路

方法一：用一个队列维护。从头开始遍历字符串，如果该字符不在队列中就加入，如果在就弹出最左边的直至不存在重复的（先入先出一直删总能删除）

方法二：左右指针法。滑动窗口的核心逻辑

「最长无重复子串」的最优解是滑动窗口（双指针）：

point1（左指针）：表示当前无重复子串的左边界；

point2（右指针）：表示当前无重复子串的右边界；

用集合记录窗口内的字符，右指针不断右移，遇到重复则左指针右移，直到没有重复的字符（即左指针一直右移到删去重复字符）同时更新最大长度。

## 代码示例
```python
#法一

def lengthOfLongestSubstring(self,s)
    from collections import deque
    queue=deque()
    length=0
    for c in s:
        while c in queue:
            queue.popleft()
        queue.append(c)
        length=max(length,len(queue))
    return length


#法二
def lengthOfLongestSubstring(self, s):
        """
        :type s: str
        :rtype: int
        """
        point1=0
        point2=0
        n=len(s)
        longestsubstr=0
        temp_set=set()
        while point2<n:
            while s[point2] in temp_set:
                temp_set.remove(s[point1])
                point1+=1
            temp_set.add(s[point2])
            longestsubstr=max(point2-point1+1,longestsubstr)
            point2+=1
        return longestsubstr
